PROJECT NAME: schoolportal
SYSTEM TYPE: Web-Based School Information & Management System
VERSION: 1.0
STATUS: Planning & Requirements Phase

============================================================
1. SYSTEM PURPOSE
============================================================
schoolportal is a web-based platform designed to help schools
digitize academic, administrative, and communication processes.
The system will support students, teachers, administrators,
parents, and school management.

Primary goals:
- Centralized student data storage
- Improve communication
- Support academic & administrative operations
- Enable secure user access
- Support multiple schools in future

============================================================
2. SYSTEM USERS (ROLES)
============================================================
1. Super Admin (System Owner)
2. School Admin (Principal / Registrar)
3. Teachers
4. Students
5. Parents / Guardians
6. Support Staff (optional)

Each user will have role-based access permissions.
